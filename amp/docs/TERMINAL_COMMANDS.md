# TDA7318 - Команди Serial терміналу

## Опис

Керування аудіопроцесором TDA7318 через Serial монітор (115200 бод).
Всі значення зберігаються в EEPROM окремо для кожного входу.

---

## Загальні команди

| Команда | Опис | Приклад |
|---------|------|---------|
| `help` | Вивести список команд | `help` |
| `status` | Поточний стан TDA7318 | `status` |

---

## Гучність (0-63)

**Діапазон:** 0 (тиша) ... 63 (максимальна гучність)  
**За замовчуванням:** 40

| Команда | Опис | Приклад |
|---------|------|---------|
| `vol <N>` | Встановити гучність N | `vol 30` |
| `vol+` | Збільшити на 1 | `vol+` |
| `vol+ <N>` | Збільшити на N | `vol+ 5` |
| `vol-` | Зменшити на 1 | `vol-` |
| `vol- <N>` | Зменшити на N | `vol- 5` |
| `v+` | Скорочено vol+ | `v+ 3` |
| `v-` | Скорочено vol- | `v- 3` |

## Перемикання входів (0-3)

**Діапазон:** 0 ... 3  
**За замовчуванням:** 0 (Bluetooth)

| Номер | Вхід | Опис |
|-------|------|------|
| 0 | Bluetooth | Bluetooth аудіо |
| 1 | Computer | Комп'ютер |
| 2 | TV Box | ТВ бокс |
| 3 | AUX | AUX вхід |

| Команда | Опис | Приклад |
|---------|------|---------|
| `input <N>` | Перемкнути вхід | `input 1` |
| `in` | Скорочено input | `in 2` |

**Примітка:** При перемиканні входу автоматично завантажуються збережені налаштування для цього входу.

## EEPROM - структура збереження

| Адреса | Дані |
|--------|------|
| 0x0000 | Магічний байт (0xA5) + поточний вхід |
| 0x0010 | Налаштування входу 0 (Bluetooth) |
| 0x0020 | Налаштування входу 1 (Computer) |
| 0x0030 | Налаштування входу 2 (TV Box) |
| 0x0040 | Налаштування входу 3 (AUX) |

**Структура налаштувань (10 байт на вхід):**
- volume (1 байт) - 0-63
- bass (1 байт) - -7 до +7
- treble (1 байт) - -7 до +7
- balance (1 байт) - -31 до +31
- gain (1 байт) - 0-3
- last_param (1 байт) - останній параметр енкодера
- reserved (3 байти)

## Швидкі комбінації

Для швидкого регулювання з клавіатури:

```
vol+ 10    → швидко збільшити гучність
vol- 10    → швидко зменшити гучність
bass+      → крок НЧ вверх
bass-      → крок НЧ вниз
treble+    → крок ВЧ вверх
treble-    → крок ВЧ вниз
bal+       → баланс вправо
bal-       → баланс вліво
bal-c      → баланс по центру
```
